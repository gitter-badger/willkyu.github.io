如何备份你的存档

2021-12-04
Pokemon

这篇文章是宝可梦正版正作卡带从 Gen1 到 Gen7 的存档提取与导入教程.
不局限于宝可梦正作，其他卡带也可以通过同样的方法提取.
如果你对备份存档感到介意，请根据自身情况取舍.

GB 与 GBC 卡带

包含的宝可梦卡带有：

  • 宝可梦 红 / 绿 / 蓝 / 黄
  • 宝可梦 金 / 银 / 水晶

这类应该只能通过特殊设备提取与导入，如 Flash Boy 、 GB Operator 等.

Flash Boy



在各电商应该都有购买方式.

GB Operator

官网地址

注意，这类卡带的存档依赖电池，如果电池没电了会导致存档丢失.

GBA 卡带

包含的宝可梦卡带有：

  • 宝可梦 红宝石 / 蓝宝石 / 绿宝石
  • 宝可梦 火红 / 叶绿

这类存档提取与导入可以通过：

  1. 使用特殊设备提取，如上述 Flash Boy 、 GB Operator.
  2. 使用 初版 NDS 或 NDS lite 配合 NDS 烧录卡与存档提取软件 GBA Backup Tool.

NDS 与 3DS 卡带 以及 3DS VC 系列

包含的宝可梦卡带有：

  • 宝可梦 珍珠 / 钻石 / 白金
  • 宝可梦 心金 / 魂银
  • 宝可梦 黑 / 白
  • 宝可梦 黑2 / 白2
  • 宝可梦 X / Y
  • 宝可梦 欧米伽红宝石／阿尔法蓝宝石
  • 宝可梦 日 / 月
  • 宝可梦 究极之日 / 究极之月
  • 宝可梦 VC 系列

这类存档提取与导入需要使用破解 3DS ，想要破解 3DS 请参见：一只火狐的杂物间.
请具备一定的破解 3DS 软件安装能力.
