解决访问GithHub速度慢的问题

2021-12-02

今天早上上 GitHub 又没上上去，去网上找了一下原因是 GitHub 的 CDN 域名遭到了 DNS 污染，导致无法连接使用 GitHub 的加速分发服务器，才使得国内访问速度很慢.

可以通过修改本地 hosts 文件来绕过 DNS 解析来加速.

  • 首先打开站长工具
  • 填入 <Github.com>
  • 将检测列表里的 TTL 值最小的 IP 记为 a.b.c.d.
